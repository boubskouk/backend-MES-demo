1. Introduction

Bienvenue dans le Systeme d'Archivage Electronique du MES. Cette application permet de stocker, organiser et gerer tous vos documents de maniere securisee et centralisee.

Qu'est-ce que cette application ?

C'est un systeme d'archivage numerique qui permet de :

  • Stocker vos documents (PDF, Word, Excel, Images, etc.)
  • Organiser les documents par categories et services
  • Partager des documents avec d'autres utilisateurs
  • Suivre l'historique de chaque document (qui a telecharge, quand, etc.)
  • Proteger les documents sensibles par verrouillage

Architecture du systeme

Super Admin → Niveau 1 (Primaire) → Niveau 2 (Secondaire) → Niveau 3 (Tertiaire)

2. Les Differents Niveaux d'Utilisateurs

L'application utilise un systeme hierarchique avec 4 niveaux d'acces. Chaque niveau a des permissions specifiques.

Niveau 0 - Super Administrateur

Role : Gestionnaire global du systeme

  • Creer et gerer TOUS les utilisateurs (niveaux 1, 2 et 3)
  • Creer et gerer les departements
  • Creer et gerer les services
  • Creer et gerer les categories de documents
  • Voir toutes les statistiques et activites
  • Superviser l'ensemble du systeme
  • Acces au tableau de bord Super Admin

Acces special

Le Super Admin se connecte via une page speciale : /super-admin-login.html

Niveau 1 - Utilisateur Primaire (Chef de Service)

Role : Responsable de l'archivage dans son service

  • Creer des dossiers et archiver des documents
  • Creer des categories de documents pour son service
  • Creer des services dans son departement
  • Partager des documents avec d'autres utilisateurs
  • Verrouiller des documents (empecher modifications)
  • Supprimer des documents
  • Telecharger des documents
  • Voir l'historique complet des documents

Responsabilite cle

Le Niveau 1 doit d'abord creer les categories et services avant que les autres niveaux puissent archiver.

Niveau 2 - Utilisateur Secondaire

Role : Utilisateur avance avec droits etendus

  • Creer des dossiers et archiver des documents
  • Partager des documents avec d'autres utilisateurs
  • Telecharger des documents
  • Modifier les dossiers qu'il a crees
  • Voir l'historique des documents

Limitations

Ne peut pas verrouiller ou supprimer definitivement des documents. Ne peut pas creer de categories ou services.

Niveau 3 - Utilisateur Tertiaire (Consultation)

Role : Utilisateur basique, principalement en lecture

  • Consulter les documents partages avec lui
  • Telecharger les documents auxquels il a acces
  • Voir ses propres dossiers

Acces limite

Ce niveau est ideal pour les utilisateurs qui ont seulement besoin de consulter des documents sans les modifier.

3. Premiere Connexion

Lorsque le Super Admin cree votre compte, il vous fournit un nom d'utilisateur et un mot de passe temporaire. Voici comment vous connecter pour la premiere fois.

1

Acceder a la page de connexion

Ouvrez votre navigateur web (Chrome, Firefox, Edge) et allez sur l'adresse de l'application.

Pour les utilisateurs normaux : https://votre-application.com/
Pour le Super Admin : https://votre-application.com/super-admin-login.html

2

Entrer vos identifiants

Saisissez le nom d'utilisateur et le mot de passe temporaire que le Super Admin vous a fournis, puis cliquez sur "Se connecter".

3

Changer votre mot de passe (OBLIGATOIRE)

A la premiere connexion, vous serez oblige de changer votre mot de passe. C'est une mesure de securite importante.

Regles pour le nouveau mot de passe

Votre nouveau mot de passe doit respecter ces criteres :

  • Minimum 8 caracteres
  • Au moins une lettre majuscule (A-Z)
  • Au moins une lettre minuscule (a-z)
  • Au moins un chiffre (0-9)
  • Au moins un caractere special (@, #, $, %, etc.)

Exemple de mot de passe valide : MonMotDePasse@2024

4

Acces au tableau de bord

Apres avoir change votre mot de passe, vous serez redirige vers votre tableau de bord. Felicitations, vous etes connecte !

Conseil de securite

Ne partagez jamais votre mot de passe avec qui que ce soit. Si vous oubliez votre mot de passe, contactez le Super Admin pour le reinitialiser.

4. Configuration Initiale (Niveau 1)

IMPORTANT - Avant d'archiver des documents

Avant de pouvoir archiver des documents, le Niveau 1 ou le Super Admin doit d'abord creer les categories et les services. Sans cela, il sera impossible d'enregistrer un dossier.

4.1 Creer une Categorie

Les categories permettent de classer les documents par type (Factures, Contrats, Rapports, etc.).

1

Acceder au menu Categories

Dans le menu lateral gauche, cliquez sur "Categories" ou cherchez l'icone de gestion des categories.

2

Cliquer sur "Nouvelle Categorie"

Cliquez sur le bouton "+ Nouvelle Categorie" pour ouvrir le formulaire de creation.

3

Remplir les informations

Nom : Le nom de la categorie (ex: "Factures", "Contrats", "Rapports annuels")
Description : Une breve description de ce que contient cette categorie

4

Enregistrer

Cliquez sur "Enregistrer" pour creer la categorie.

Exemples de categories courantes

  • Factures - Documents de facturation
  • Contrats - Accords et conventions
  • Rapports - Rapports d'activite
  • Correspondance - Lettres et courriers
  • RH - Documents ressources humaines
  • Technique - Documentation technique

4.2 Creer un Service

Les services representent les differentes unites organisationnelles (Service Comptabilite, Service Technique, etc.).

1

Acceder au menu Services

Dans le menu lateral, cliquez sur "Services".

2

Cliquer sur "Nouveau Service"

Cliquez sur le bouton "+ Nouveau Service".

3

Remplir les informations

Nom du service : Le nom du service (ex: "Comptabilite", "Informatique")
Departement : Selectionnez le departement auquel appartient ce service

4

Enregistrer

Cliquez sur "Enregistrer" pour creer le service.

Configuration terminee !

Une fois les categories et services crees, tous les utilisateurs (Niveau 1, 2 et 3) peuvent commencer a archiver des documents.

5. Archiver un Document

L'archivage se fait en deux etapes : d'abord creer un dossier, puis y ajouter des documents (fichiers).

5.1 Creer un Dossier

1

Cliquer sur "Nouveau Dossier"

Sur le tableau de bord, cliquez sur le bouton "Nouveau Dossier".

2

Remplir les informations du dossier

Titre du dossier : Un nom descriptif (ex: "Factures Janvier 2024")
Description : Details supplementaires sur le contenu
Categorie : Selectionnez la categorie appropriee
Service : Selectionnez votre service
Departement : Selectionnez votre departement

Attention

Si les listes Categorie ou Service sont vides, c'est que le Niveau 1 ou le Super Admin ne les a pas encore creees. Contactez votre responsable.

3

Ajouter des fichiers

Cliquez sur la zone "Deposer des fichiers ici" ou faites glisser vos fichiers.

Formats acceptes : PDF, Word (.doc, .docx), Excel (.xls, .xlsx), Images (PNG, JPG), etc.
Taille maximale : 50 Mo par fichier

4

Enregistrer le dossier

Cliquez sur "Enregistrer" ou "Creer le dossier". Le dossier et ses documents sont maintenant archives.

5.2 Ajouter des Documents a un Dossier Existant

1

Ouvrir le dossier

Sur le tableau de bord, trouvez le dossier et cliquez dessus pour l'ouvrir.

2

Ajouter un fichier

Cliquez sur "Ajouter un document" et selectionnez le fichier a ajouter.

Document archive !

Votre document est maintenant stocke de maniere securisee dans le systeme. Vous pouvez le retrouver a tout moment via la recherche ou en parcourant vos dossiers.

6. Gestion des Documents

6.1 Consulter un Document

Cliquez sur le bouton "Consulter" (icone oeil) pour visualiser le document directement dans votre navigateur.

6.2 Telecharger un Document

Cliquez sur le bouton "Telecharger" (icone fleche vers le bas) pour sauvegarder le document sur votre ordinateur.

Tracabilite

Chaque telechargement est enregistre dans l'historique du document. Le Niveau 1 et le Super Admin peuvent voir qui a telecharge quoi et quand.

6.3 Partager un Document

Le partage permet a d'autres utilisateurs d'acceder a un document.

1

Cliquer sur "Partager"

Cliquez sur le bouton "Partager" du document.

2

Selectionner les destinataires

Choisissez les utilisateurs avec qui vous souhaitez partager le document.

3

Confirmer le partage

Cliquez sur "Partager" pour confirmer.

6.4 Verrouiller un Document (Niveau 1 uniquement)

Le verrouillage empeche toute modification ou suppression du document.

Attention

Seul le Niveau 1 qui a verrouille le document peut le deverrouiller. Utilisez cette fonction pour les documents sensibles ou definitifs.

6.5 Voir l'Historique d'un Document

Cliquez sur le bouton "Historique" pour voir les 20 dernieres actions effectuees sur le document :

  • Qui a cree le document
  • Qui l'a telecharge et quand
  • Qui l'a partage et avec qui
  • Qui l'a verrouille/deverrouille

6.6 Supprimer un Document (Niveau 1 uniquement)

La suppression deplace le document vers la corbeille. Le Super Admin peut le restaurer si necessaire.

Attention

Un document verrouille ne peut pas etre supprime. Il faut d'abord le deverrouiller.

7. Questions Frequentes (FAQ)

?

Je ne peux pas creer de dossier, les categories sont vides

C'est normal si aucune categorie n'a ete creee. Contactez votre responsable Niveau 1 ou le Super Admin pour qu'il cree les categories necessaires.

?

J'ai oublie mon mot de passe

Contactez le Super Admin. Il peut reinitialiser votre mot de passe depuis son tableau de bord.

?

Je ne vois pas certains documents

Vous ne voyez que les documents qui vous ont ete partages ou ceux que vous avez crees. Si vous avez besoin d'acces a d'autres documents, demandez a leur proprietaire de les partager avec vous.

?

Le document ne se telecharge pas

Verifiez votre connexion internet. Si le probleme persiste, il peut y avoir un probleme avec le fichier. Contactez le support technique.

?

Je ne peux pas modifier un dossier

Si le dossier est verrouille (icone cadenas), vous ne pouvez pas le modifier. Seul le Niveau 1 qui l'a verrouille peut le deverrouiller.

?

Comment contacter le support ?

Pour toute assistance technique, contactez le Super Admin ou l'equipe informatique du MES.
